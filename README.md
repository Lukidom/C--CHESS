# C++ Chess Board (Starter)

This project is a simple C++ chess-board starter program.

## What It Does

- Creates an 8x8 chess board
- Sets up the standard starting position
- Prints the board to the console
- Demonstrates one move (`e2 -> e4`)

## File

- `main.cpp` - all source code

## Build

Use the VS Code task:

- `Build C++ (auto compiler)`

Or compile manually with `g++`:

```bash
g++ -std=c++17 -Wall -Wextra -pedantic main.cpp -o main.exe
```

## Run

Use the VS Code task:

- `Run main.exe`

Or run manually:

```bash
./main.exe
```
